# Hướng dẫn cài đặt Backy2 để Backup Ceph RBD

## 1. Giới thiệu

Backy2 là một công cụ backup dạng **block-level backup**, thường được dùng để backup:

* VM disk
* Block device
* Ceph RBD image

Backy2 hoạt động theo cơ chế:

* chia dữ liệu thành block (mặc định 4MB)
* deduplication theo checksum
* lưu metadata vào database (SQLite hoặc PostgreSQL)

Trong tài liệu này sẽ trình bày cách:

* Cài đặt Backy2 trên Ubuntu
* Kết nối với cluster Ceph
* Backup RBD image
* Kiểm tra backup


# 2. Kiểm tra trạng thái Ceph Cluster

Trước khi thực hiện backup cần đảm bảo cluster Ceph hoạt động bình thường.

### Kiểm tra trạng thái cluster

```bash
ceph -s
```

### Kiểm tra trạng thái placement group

```bash
ceph pg stat
```

### Kiểm tra OSD tree

```bash
ceph osd tree
```

### Kiểm tra pool

```bash
ceph osd pool ls detail
```

### Theo dõi trạng thái cluster realtime

```bash
watch -n 5 ceph -s
```

hoặc

```bash
watch -n 5 ceph pg stat
```

# 4. Cài đặt Backy2

Tải package Backy2 từ GitHub:

```bash
wget https://github.com/wamdam/backy2/releases/download/2.13.8/backy2_2.13.8-1_amd64.deb
wget https://github.com/wamdam/backy2/releases/download/2.13.8/backy2_2.13.8_all.deb
```

Cài đặt package:

```bash
sudo dpkg -i backy2_*.deb
```

Cài đặt các dependency còn thiếu:

```bash
sudo apt-get -f install
```

 

# 5. Cài đặt các thư viện Python cần thiết

Backy2 yêu cầu một số thư viện Python mới hơn so với repo Ubuntu.

Cài pip:

```bash
apt install python3-pip
```

Cài các thư viện cần thiết:

```bash
pip3 install pycryptodome zstandard
```

Cài thư viện hỗ trợ Ceph RBD:

```bash
sudo apt install python3-rbd python3-rados
```

Nếu sử dụng PostgreSQL:

```bash
sudo apt install python3-psycopg2
```

 

# 6. Tạo user và thư mục lưu backup

Tạo user dành riêng cho backy2:

```bash
sudo useradd -r -s /usr/sbin/nologin backy
```

Tạo thư mục lưu dữ liệu backup:

```bash
sudo mkdir -p /var/lib/backy2/data
```

Cấp quyền cho user backy:

```bash
sudo chown -R backy:backy /var/lib/backy2
```

 

# 7. Cấu hình Backy2

File cấu hình:

```bash
/etc/backy.cfg
```

Mở file để chỉnh sửa:

```bash
vi /etc/backy.cfg
```

### Cấu hình nơi lưu metadata

Ví dụ dùng SQLite:

```
engine: sqlite:////var/lib/backy2/backy.sqlite
```

### Cấu hình nơi lưu dữ liệu backup

```
path: /var/lib/backy2/data
```

 

# 8. Tạo encryption key

Backy2 yêu cầu encryption key để mã hóa dữ liệu.

Tạo key:

```bash
openssl rand -hex 32
```

Ví dụ kết quả:

```
c1a0f2c1c3f5c9a3b1e5c0a7b0c8a4c1c3e5f7a9b2d4c6e8f0a1b3c5d7e9f1a2
```

Thêm vào file cấu hình:

```
encryption_key: <generated_key>
encryption_version: 1
```

 

# 9. Khởi tạo database Backy2

Sau khi cấu hình xong, khởi tạo database:

```bash
backy2 initdb
```

Kiểm tra:

```bash
backy2 ls
```

Nếu thành công sẽ không có lỗi.

# 11. Cài đặt Ceph client

Để backup RBD cần cài ceph client.

```bash
sudo apt install ceph-common -y
```

Kiểm tra thư mục cấu hình:

```bash
cd /etc/ceph/
ls
```

Thông thường sẽ có:

```
ceph.conf
ceph.client.backup.keyring
```

 

# 12. Kiểm tra kết nối Ceph

Test kết nối cluster bằng user backup:

```bash
ceph -n client.backup \
--keyring /etc/ceph/ceph.client.backup.keyring -s
```

Nếu thành công sẽ hiển thị trạng thái cluster.

 

# 13. Kiểm tra RBD image

Liệt kê các image trong pool:

```bash
rbd -n client.backup \
--keyring /etc/ceph/ceph.client.backup.keyring \
-p images ls
```

Ví dụ:

```
vol1
vol2
vol6
```

 

# 14. Backup Ceph RBD Image

Backup image `vol1` trong pool `images`:

```bash
backy2 backup rbd://images/vol1 "backup vol1"
```

Backup image `vol6`:

```bash
backy2 backup rbd://images/vol6 "backup vol6"
```


 

# 16. Một số lệnh quản trị Backy2

### Kiểm tra tính toàn vẹn backup

```bash
backy2 scrub <backup_uuid>
```

### Restore backup

```bash
backy2 restore <backup_uuid> file://restorefile
```

### Xóa backup

```bash
backy2 rm <backup_uuid>
```

### Dọn dẹp block không sử dụng

```bash
backy2 cleanup
```
